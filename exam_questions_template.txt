Exam Questions Template Format
================================

This file shows the correct format for importing questions into an exam.

Columns required:
1. Question - The question text
2. OptionA - Option A for the question
3. OptionB - Option B for the question
4. OptionC - Option C for the question
5. OptionD - Option D for the question
6. CorrectAnswer - The correct option (A, B, C, or D)
7. Points (optional) - Points for the question (default is 1)
8. Explanation (optional) - Explanation for the correct answer

Example:
Question,OptionA,OptionB,OptionC,OptionD,CorrectAnswer,Points,Explanation
"What is the capital of France?","London","Berlin","Paris","Madrid","C","1","Paris is the capital of France"
"What is 2 + 2?","3","4","5","6","B","1","Basic addition"
"Which planet is known as the Red Planet?","Venus","Mars","Jupiter","Saturn","B","1","Mars appears red due to iron oxide"

To create an Excel file:
1. Open Microsoft Excel or Google Sheets
2. Copy the above format into the spreadsheet
3. Save as .xlsx or .xls file
4. Use this file to import questions into your exam
